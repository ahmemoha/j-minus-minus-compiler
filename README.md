## Author
Name: Ahmed Mohamed

## Description
This project implements a lexical analyzer/scanner for the J-- language using Python 3 and ANTLR4.
It reads a source file and outputs the stream of tokens to standard output.

## Build Instructions
Run `make` to generate the ANTLR4 lexer files.

## Usage
python3 main.py <input_file>
